CoreCMS is a powerful and feature-rich Content Management System (CMS) built on the ASP.NET Core framework.

Getting Started with CoreCMS

Welcome to CoreCMS, an ASP.NET Core Content Management System built with a focus on Domain-Driven Design (DDD). This guide will walk you through the steps to set up and start using CoreCMS for your content management needs. Prerequisites

Requirement

Before getting started, ensure that you have the following prerequisites installed on your system:

  • .NET Core SDK (version X.X or later)
  • Visual Studio or any code editor of your choice

Installation

Follow these steps to install and set up CoreCMS:

Clone the CoreCMS repository from GitHub by running the following command in your terminal or command prompt:

  
    git clone https://github.com/your-username/CoreCMS.git

Open the project in your preferred code editor.

Restore the required NuGet packages by running the following command in the project's root folder:

  
    dotnet restore

Next, create the database required for CoreCMS. Open the appsettings.json file and update the database connection string with your preferred database provider and connection details.

Apply the necessary database migrations by running the following command:

  
    dotnet ef database update

Once the migrations are applied successfully, build the solution by running:

  
    dotnet build

Finally, start the CoreCMS application by running the following command:

  
    dotnet run

Congratulations! CoreCMS is now up and running on your local machine.

Use the default administrator credentials to log in:

Username: admin
Password: password

Happy content managing with CoreCMS!
