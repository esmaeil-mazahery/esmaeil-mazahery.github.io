HTML Document

  
    <!DOCTYPE html>
    <html lang="en-US">
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <h1>This is a Heading</h1>
            <p>This is a paragraph.</p>
        </body>
    </html>

HTML is Not Case Sensitive

lang Declare the language of the web page to be used by search engines and browsers using ISO 639-1 language codes.

URL

Absolute URL A complete address, including the domain name, which may represent an external source.

Relative URL An address, without domain , which hosted within the website. If begin without / slash Relative to current page. If begin with / slash Relative to domain.
