Target project : Content management system with users and roles
VS Code (https://code.visualstudio.com/download) Dotnet 5 SDK (https://dotnet.microsoft.com/download) checking our dotnet SDK dotnet --version install the entity framework tool dotnet tool install --global dotnet-ef

Create Empty Solution 

  
    dotnet new sln --output Blog
    cd Blog

Create Projects

create 4 project in solution with this names

  • Infrastructure it contains all public codes and use by all other project 
          
        dotnet new classlib --output Blog.Infrastructure
  • DataLayer this layer connect to database and hold model, migration and mapping of data in database 
          
        dotnet new classlib --output Blog.DataLayer
  • ServiceLayer this layer contains services and handle all functionality of program 
          
        dotnet new classlib --output Blog.ServiceLayer
  • WebApplication this layer contains user interface all request are handeled in this project 
          
        dotnet new mvc --output Blog.WebApplication
